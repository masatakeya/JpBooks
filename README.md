# Obsidianプラグイン「JpBooks」基本仕様書

**バージョン**: 1.1  
**作成日**: 2026-03-03

---

## 1. 開発の目的

日本の書籍情報を、複数のデータソースから効率的かつ安定的に取得する。特に新刊情報の反映遅延を解消し、正確な書誌データを基にした読書ノート作成の自動化を実現することを目的とする。

---

## 2. 主要機能

| 機能 | 概要 |
|------|------|
| 書籍検索 | ISBN（10桁/13桁）または書名キーワードによる検索 |
| ハイブリッド・データ取得 | 複数のAPIをフォールバック方式で自動参照 |
| ノート自動生成 | 取得したメタデータをテンプレートに流し込みノートを生成 |

---

## 3. ハイブリッド・データ取得アーキテクチャ

データの信頼性と即時性を確保するため、以下のフォールバック方式でリクエスト処理を行う。

### フォールバック方式の定義

openBDで書籍情報が**取得できた場合**はその結果を使用する。openBDで情報が**取得できなかった場合のみ**、NDL Searchに問い合わせる。

### 第一優先: openBD API

- **目的**: 新刊情報の高速取得
- **処理**: 出版社による最新データおよび近刊情報の取得
- **エンドポイント**: `https://api.openbd.jp/v1/get`

### 第二優先: 国立国会図書館サーチ (NDL Search) API

- **目的**: 網羅性の確保
- **処理**: openBDで取得できなかった既刊本・学術書の書誌データを取得
- **エンドポイント**: `https://iss.ndl.go.jp/api/opensearch`

---

## 4. 取得・出力項目（メタデータ）

| 項目名 | 内容 | 出力形式（例） |
|--------|------|----------------|
| Title | 書名・副題 | `title: "書籍名"` |
| Author | 著者名・カンマ区切り文字列 | `author: "著者1, 著者2"` |
| Authors | 著者名の配列 | `authors: ["著者1", "著者2"]` |
| Publisher | 出版社 | `publisher: "○○出版"` |
| PubDate | 出版年月日 | `published: 2026-01-01` |
| ISBN | ISBN-13 | `isbn: "9784..."` |
| Description | 内容紹介・目次 | ノート本文の引用セクション |

## 5. ユーザーインターフェース

### コマンドパレット

| コマンド | 動作 |
|----------|------|
| `JpBooks: Search and Insert` | 検索した書誌情報をカーソル位置に挿入 |
| `JpBooks: Create New Note` | 書籍名で新しいノートを作成 |

### 検索モーダル

1. キーワード（ISBN または書名）を入力
2. 検索結果が複数件の場合、候補を一覧表示
3. ユーザーが選択した書籍の情報を取得・出力

### 設定画面 (Settings)

| 設定項目 | 内容 |
|----------|------|
| APIの優先順位 | 使用するAPIの有効/無効および順序の変更 |
| 出力テンプレート | `{{title}}`形式によるテンプレートのカスタマイズ |
| ファイル名フォーマット | 新規ノートのファイル名形式を指定（デフォルト: `{{title}}`） |
| 保存先フォルダ | 新規ノートの保存先パスを指定 |

### ファイル名フォーマットの例

| フォーマット | 出力例 |
|-------------|--------|
| `{{title}}` | `UIデザインの教科書.md` |
| `{{title}} - {{author}}` | `UIデザインの教科書 - 原田秀司.md` |
| `{{DATE:YYYYMMDD}}_{{title}}` | `20260303_UIデザインの教科書.md` |

### テンプレート形式

Obsidianコアプラグインと統一した`{{変数名}}`形式を採用する。

**使用可能な変数:**

| 変数 | 内容 |
|------|------|
| `{{title}}` | 書名 |
| `{{author}}` | 著者名（複数の場合はカンマ区切り文字列） |
| `{{authors}}` | 著者名の配列（複数著者を個別に扱う場合） |
| `{{publisher}}` | 出版社 |
| `{{published}}` | 出版年月日 |
| `{{isbn}}` | ISBN-13 |
| `{{description}}` | 内容紹介 |
| `{{DATE}}` | ノート作成日（デフォルト: YYYY-MM-DD） |
| `{{DATE:YYYYMMDD}}` | ノート作成日（書式指定可） |

**テンプレート例:**

```markdown
---
title: "{{title}}"
author: {{author}}
publisher: "{{publisher}}"
published: {{published}}
isbn: "{{isbn}}"
tags: [読書ノート]
---

## 内容紹介

{{description}}

## メモ

```
